Latest Projects

Tic Tac Toe

Created a Tic Tac Toe game in JavaScript by organizing game logic into modules and factories, allowing players to interact with a dynamic gameboard, place markers, check for a winner or tie, and restart the game with an updated interface.

Library App

Developed a Library app by creating Book and Library classes in JavaScript, enabling users to add books with attributes like title, author, and read status, and dynamically display them in a table with functionality to mark books as read.

Weather Api

Developed a Weather App using a weather API that allows users to search locations, toggle between Fahrenheit and Celsius, save and remove favorite locations, and display real-time weather data with dynamic styling and additional features like loading indicators and weather-related GIFs.

Memory card

Built a Memory Card game in React using lifecycle methods, props, and state, where users click items, track their selections, and compete for the best score while randomizing items after each click.

Api Project

This API project is built using Express and MySQL to manage a characters database with full CRUD functionality (GET, POST, PUT, DELETE). The database stores character details, and all server requests are logged in a separate table.

Pokemon

This project is a Pokémon Index built with React and React Router, allowing users to browse a list of Pokémon and view detailed information about each one. It fetches data from a Pokémon API, displaying it dynamically using components like PokemonList and PokemonDetails. The app is structured with routing for navigation, CSS for styling, and GitHub Pages for deployment.
